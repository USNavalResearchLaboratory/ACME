4.2. Communications Coordinator¶

class acme.coordinators.CommsCoordinator.CommsCoordinator(thePlatforms, theNumFrequencyBins, theMediumAccessMethod)[source]¶

Implements the medium access control (MAC) layer.

Coordinates incoming units of information from multiple Communications Platforms and decides how to place these information units into the RF Environment for transmission.

Warning

This class is typically instantiated automatically by the Environment and is generally not instantiated directly by the user.

The tuple containing the Communications Platforms that this Coordinator manages.
